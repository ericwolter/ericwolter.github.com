Position Based Fluids

Description

Realtime GPU Water Simulation using position based dynamics. This approach is ideally suited for a large number of particles and to behave physically plausible. The implementation can simulate over 50.000 particles with 60 FPS on a mid-range GPU.

Original work by Macklin et al.

Technologies

  • OpenCL
  • OpenGL
  • C++
